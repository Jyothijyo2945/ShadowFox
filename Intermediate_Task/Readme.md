# AQI Analysis Project

## ShadowFox Data Science Internship - Intermediate Level Task

This project is completed as part of the ShadowFox Data Science Internship Intermediate Level Task.

---

# Project Objective

The main objective of this project is to analyze air pollution data and identify pollution trends using Python data analysis and visualization libraries.

The project focuses on understanding pollutant behavior, relationships between pollutants, and overall pollution trends through graphical analysis.

---

# Dataset Information

The dataset contains air pollution related parameters such as:

- CO (Carbon Monoxide)
- NO (Nitric Oxide)
- NO2 (Nitrogen Dioxide)
- O3 (Ozone)
- SO2 (Sulfur Dioxide)
- PM2.5
- PM10
- NH3 (Ammonia)

The dataset also contains date information for trend analysis.

---

# Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

# Analysis Performed

## 1. Data Loading
The dataset was loaded using Pandas.

## 2. Data Cleaning
Missing values were identified and handled.

## 3. Exploratory Data Analysis (EDA)
Basic statistical analysis was performed to understand the dataset.

## 4. Correlation Analysis
Relationships between pollutants were analyzed using a heatmap.

## 5. Distribution Analysis
PM2.5 pollutant distribution was analyzed using histogram visualization.

## 6. Trend Analysis
Pollution trends were visualized using line plots.

## 7. Pollutant Comparison
Average pollutant levels were compared using bar charts.

## 8. Scatter Plot Analysis
Relationships between PM2.5 and PM10 were visualized using scatter plots.

---

# Visualizations Created

- Correlation Heatmap
- PM2.5 Distribution Histogram
- PM2.5 Trend Analysis
- Pollutant Comparison Bar Chart
- Scatter Plot (PM2.5 vs PM10)

# Key Insights

- PM2.5 and PM10 are major contributors to air pollution.
- Pollutants show positive correlation with each other.
- Pollution levels vary across observations and time.
- Heatmap analysis helps identify strong pollutant relationships.
- Data visualization provides better understanding of pollution patterns.
